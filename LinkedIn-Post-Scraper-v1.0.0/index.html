Privacy Policy — Automatic Post Scraper

Last updated: June 2026

This extension does not collect, store, transmit, or share any personal user data with any third party or external server.

What the extension accesses

  • Publicly visible post content on LinkedIn and Twitter/X pages that the user is actively viewing
  • Data is processed locally on the user's device only
  • Temporarily stored in chrome.storage.local solely to maintain scraping state between scroll cycles
  • All collected data is downloaded directly to the user's own device as an Excel or CSV file
  • No data is sent to any external server or third party

The extension does NOT

  • Track browsing history
  • Access passwords or credentials
  • Collect personal information about the user
  • Transmit any data outside the user's device
  • Sell or share data with advertisers or third parties
